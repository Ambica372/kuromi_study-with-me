Flow Study AI

A gamified study system designed to make studying addictive instead of exhausting.

---

🚀 Concept

Most study tools fail because they depend on discipline.

This project flips that.

Instead of forcing focus, it builds a system where:

- you get rewarded for showing up
- progress is visible
- studying feels like a game loop

---

🧠 Core Idea

Study → Earn → Upgrade → Customize → Repeat

The goal is simple:
Make users want to come back.

---

✨ Features

⏱ Pomodoro System

- Custom study + break timer
- Session tracking
- Visual progress

🪙 Reward System

- Coins + XP for each session
- Level progression
- Instant feedback after completion

🐾 Kuromi Virtual Pet

- Reacts to your activity
- Gets happy when you study
- Adds emotional attachment to progress

🛍 Shop System

- Buy items using earned coins
- Categories: food, furniture, accessories, skins
- Equip and customize your setup

🏠 Room Builder

- Drag, resize, and rotate items
- Place objects anywhere
- Build your own study space

🃏 Flashcards

- Create your own cards
- Flip interaction
- Download as interactive HTML

🎵 Focus Environment

- Background music (lofi, ambient, rain)
- Study-friendly atmosphere

🔔 Reminder System

- In-app nudges to return and study
- Behavior-driven notifications

---

🎯 Purpose

This is not a productivity tool.

It’s a behavior system designed to:

- reduce resistance to starting
- build consistency
- make studying feel rewarding
